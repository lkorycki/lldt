# lldt
Experiments for: *Streaming Decision Trees for Lifelong Learning*.

### Code
- HT/HT-AE, IRF/IRF-AE, BAG, RSP: *learners/ht.py*, *learners/irf.py*

- Evaluation: *benchmark/lldt_runner.py*

### Packages
```
pip install wheel
pip install -r requirements.txt
```

### Run

```
python main.py
```
